# Webpack ES6 + SCSS Boilerplate

### Example
  ![Example](example.gif)

### Run dev server
``
yarn start
``

### Run css and js lint
``
yarn run lint
``

### Run scss lint
``
yarn run lint:scss
``

### Run js lint
``
yarn run lint:js
``

### Run fix js lint errors
``
yarn run lint:js:fix
``

### Run build
``
yarn run build
``

### Run build for prod
``
yarn run build:prod
``
